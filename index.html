Tentang Saya

Nama saya Ahmad Mufid. Sekarang saya sedang menempuh pendidikan S1 di Universitas Sains Al-Qur'an (UNSIQ) Jawa Tengah di Wonosobo mengambil program pendidikan Teknik Informatika, Fakultas Teknik dan Ilmu Komputer. Saya adalah seorang peminat dunia programming yang sedang belajar dan menekuni C++ dan Java di bidang Desktop, dan JavaScript dan PHP di bidang Web. Bisa dibilang saya merupakan seorang pemula atau biasa disebut newbie, namun saya mencoba untuk berusaha mempelajari semuanya lebih banyak dari sebelumnya yaitu dalam programming.

Bagi saya dua bidang tersebut (Web Programming) dan (Desktop Programming) adalah bidang-bidang yang semuanya unik dan memiliki sisi edukasi yang tidak dapat -untuk saat ini, bagi pemula seperti saya- untuk ditinggalkan salah satunya. Karena dengan mempelajari keduanya, mampu memberikan pemahaman yang menyeluruh relatif lebih cepat daripada hanya membaca satu bidang saja.

Melalui belajar (Desktop Programming) misalnya, saya dapat memehami -atau paling tidak mendapat gambaran umumnya- konsep dari Pemrograman Berorientasi Objek (OOP) lebih cepat karena sifat dari Java sendiri yang akan error kalau kita tidak menaati aturan OOP-nya. Otomatis ketika program kita berjalan lancar maka saat itu juga OOP kita telah benar.

Kemudian juga melalui belajar C++ OOP saya bisa memahami tentang konsep Polymorphism dengan lebih cepat daripada saya memahami lewat bahasa lain. Walaupun syntax C++ sendiri adalah sangat rumit jika sudah merambah OOP.

Kemudian dalam bidang Web Programming saya lebih cepat memahami implementasi database yaitu dengan materi CRUD (Create / membuat, Read / membaca, Update / mengubah, Delete / menghapus). Melalui mempelajari dunia Web Programming ini juga saya menemukan implementasi-implementasi tentang materi sulit tingkat atas dengan lebih cepat misalnya tentang framework dan topik-topik lain yang memang dalam Web Programming terkesan pengimplementasi materi tingkat atas merupakan hal yang umum dan populer.

Memang pada akhirnya kita tidak dapat mengambil semua hal yang ada di dunia ini dan berfokus pada satu bidang spesifik merupakan hal yang seharusnya dilakukan. Maka dari itu saat ini saya mencoba melebihkan fokus saya pada bidang Web Programming untuk persiapan awal memasuki dunia pekerjaan dengan hanya sekedarnya dalam menyelami dan mempelajari dunia Desktop Programming dengan pertimbangan dan asumsi bahwa bidang tersebut sekarang seperti terlihat sangat dicari, walaupun tidak selalu menunjukkan juga dengan besar imbalannya melebihi bidang Desktop Programming.
